# ⚡ 快速开始指南（5分钟部署）

## 🎯 最简单的部署方式

### 第一步：推送到 GitHub（2分钟）

```bash
# 如果还没有 Git 仓库
git init
git add .
git commit -m "Add upload feature"

# 在 GitHub 创建仓库后
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

### 第二步：部署到 Vercel（2分钟）

1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "Add New Project"
4. 选择你的仓库，点击 "Import"
5. 直接点击 "Deploy"（不需要修改任何设置）
6. 等待 1-2 分钟，复制部署地址

### 第三步：配置编辑器（1分钟）

1. 打开游戏，进入关卡编辑器
2. 点击 "配置 API"
3. 输入：`https://你的项目名.vercel.app/api/levels`
4. 完成！

### 第四步：测试上传

1. 编辑一个关卡
2. 点击 "保存到内存"
3. 点击 "上传到服务器"
4. 看到 "✅ 上传成功" 就完成了！

---

## 💡 提示

- **基础版**：数据存储在内存，重启后丢失（适合测试）
- **生产环境**：使用 Supabase 版本（见 `部署教程.md`）
